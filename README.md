# Online Black Jack

A simple browser-based Black Jack game built as part of a course in web development.

The project includes:
- User accounts
- Login system
- Persistent balance using localStorage
- Betting system
- Black Jack gameplay against a dealer

---

## Technologies

- HTML
- CSS
- JavaScript (ES Modules)
- localStorage

---

## How to Run

1. Download or clone the project
2. Open the project folder in VS Code
3. Start Live Server
4. Open the browser

## Game Rules

- Goal is to get as close to 21 as possible without going over
- Number cards are worth their number
- Face cards are worth 10
- Ace counts as 11 or 1 depending on the hand score
- Dealer draws until reaching at least 17
- If player goes over 21, dealer wins
